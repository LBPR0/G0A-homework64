CSS Rule (CSS წესი): ეს არის სრული ინსტრუქცია, რომელიც განსაზღვრავს, თუ როგორ უნდა გამოიყურებოდეს HTML ელემენტი. შედგება Selector-ისა და Declaration Block-ისგან. Selector (სელექტორი): მიუთითებს, რომელ HTML ელემენტზე გავრცელდება წესი. მაგ: p ნიშნავს ყველა პარაგრაფს. Declaration (დეკლარაცია): აღწერს, რა სტილი უნდა მიენიჭოს ელემენტს. შედგება Property-სა და Value-სგან. მაგ: color: red; Property (თვისება): განსაზღვრავს, რა ასპექტს ვცვლით (მაგ: ფერი, ზომა). მაგ: color Value (მნიშვნელობა): განსაზღვრავს, რა მნიშვნელობას ვანიჭებთ თვისებას. მაგ: red

ეს არის პარაგრაფი, რომელსაც აქვს Internal CSS სტილები.
